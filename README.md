# react-dva-demo
项目代码大部分来自[《12 步 30 分钟，完成用户管理的 CURD 应用 (react+dva+antd)》](https://github.com/dvajs/dva/tree/master/packages/dva-example-user-dashboard)，个人对其中的一些代码进行了优化

## Getting Started
Install dependencies.

```bash
$ npm install
```

Start server.

```bash
$ npm run start
```

Build 
```bash
$ npm run build
```
